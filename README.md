# Calculator Project

## Description
Ce projet est une calculatrice développée en Python permettant d'effectuer plusieurs opérations mathématiques.

## Fonctionnalités
- Addition
- Soustraction
- Multiplication
- Division
- Modulo (Mod)
- Carré d'un nombre
- Factorielle d'un nombre
- Moyenne
- Valeur absolue
- Puissance
- PGCD (Plus Grand Commun Diviseur)
- Racine carrée
- Vérifier si un nombre est pair


## Outils utilisés
- Python
- Visual Studio Code (VS Code)
- Git
- GitHub
