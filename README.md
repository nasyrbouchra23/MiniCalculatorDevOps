# Calculator Project

## Description
Ce projet est une calculatrice développée en Python permettant d'effectuer plusieurs opérations mathématiques.

## Fonctionnalités
- Addition
- Soustraction
- Multiplication
- Division
- Modulo
- Carré d'un nombre
- Factorielle d'un nombre

## Outils utilisés
- Python
- Visual Studio Code (VS Code)
- Git
- GitHub
